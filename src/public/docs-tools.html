API Endpoints

Explore the various API endpoints provided by AlexaAPI. Below is the documentation on how to use each endpoint.

1. Youtube transcript

Transcript any video from youtube

Get /api/transcript?url={url}

Parameters:

  • url: Enter the URL of the YouTube video you want to transcript.

Example Request:

axios.get('{endpoint}/api/transcript?url=https://youtu.be/IpFX2vq8HKw?si=T9X_I1I0azjQeUzK'
).then(response => {
  console.log(response.data);
}).then(error => {
  console.log(error)
})

Example Response:

{
    "status": true,
    "result": " result text"
}
